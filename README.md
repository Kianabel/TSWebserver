## Installation

### 1. Clone the repository:

```sh
git clone https://github.com/Kianabel/TSWebserver.git
cd TSWebserver
```

### 2. Install dependencies

```sh
npm install
```

### 3. Build and Start the Server

Build the JavaScript files using the script:
```sh
npm run build
```

or using tsc:
```sh
npx tsc
```

Now run the server:
```sh
npm run start
```
